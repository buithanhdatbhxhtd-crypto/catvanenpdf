Công Cụ Xử Lý PDF Chuyên Nghiệp - Mr Đạt

Chào mừng bạn đến với ứng dụng xử lý file PDF dành riêng cho công tác nghiệp vụ Bảo hiểm xã hội. Ứng dụng này được tối ưu hóa để xử lý các file hồ sơ siêu nặng, giúp việc lưu trữ và gửi file qua Zalo/Email trở nên dễ dàng hơn.

✨ Tính năng chính:

Nén PDF Siêu Cấp: Sử dụng công nghệ Ghostscript để ép dung lượng file từ hàng chục MB xuống dưới 5MB mà vẫn đảm bảo độ đọc rõ của văn bản.

Cắt PDF Thông Minh: Chia nhỏ file PDF nặng thành các phần bằng nhau (ví dụ: cắt 1 file 12MB thành 3 file 4MB) để đáp ứng các yêu cầu upload hệ thống.

🛠️ Cấu trúc thư mục GitHub:

Để chạy trên web, anh Đạt hãy đảm bảo GitHub có đủ các file sau:

pdf_tools_app.py: File mã nguồn chính (đã hợp nhất tính năng nén và cắt).

requirements.txt: Khai báo thư viện (pymupdf, streamlit).

packages.txt: Khai báo để server tự cài Ghostscript.
